
All animations auto-initialize on page load:

```javascript
// Automatically initializes all animations
```

You can also initialize specific animations:

```javascript

// Initialize just black hole animations
```

## üéØ Best Practices

### 1. Container Size

Ensure animation containers have explicit dimensions:

```css
.pulsar-animation {
    width: 100%;
    height: 100%;
    min-height: 200px;
    position: relative;
    overflow: hidden;
}
```

### 2. Memory Management

Animations automatically clean up removed elements. To stop an animation:

```javascript
// Clear container to stop animation
const container = document.querySelector('.pulsar-animation');
container.innerHTML = '';
```

### 3. Multiple Instances

Each container gets its own independent animation:

```html
<!-- Multiple pulsars -->
<div class="pulsar-animation"></div>
<div class="pulsar-animation"></div>
<div class="pulsar-animation"></div>
```

### 4. Reduced Motion

Respect user preferences:

```css
@media (prefers-reduced-motion: reduce) {
    .pulsar-animation,
    .blackhole-animation {
        display: none; /* Or show static version */
    }
}
```

### Combining Animations

```javascript

completeAnimations.initPulsarAnimations();
completeAnimations.initCosmicDustAnimations();
```

### Custom Animations

Create your own following the pattern:

```javascript
initMyCustomAnimation() {
    document.querySelectorAll('.my-animation').forEach(container => {
        const createEffect = () => {
            const element = document.createElement('div');
            element.style.cssText = `
                position: absolute;
                width: 10px;
                height: 10px;
                background: rgba(255, 100, 100, 0.8);
                border-radius: 50%;
            `;
            container.appendChild(element);

            // Animate...
            let x = 0;
            const animate = () => {
                x += 1;
                element.style.left = x + '%';
                if (x < 100) requestAnimationFrame(animate);
                else element.remove();
            };
            animate();
        };

        setInterval(createEffect, 1000);
    });
}
```
